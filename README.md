# 排班日历专业版 (Shift Scheduler Professional)

这是一个基于tkinter的排班管理系统，具有以下主要功能：

## 核心功能

1. **班次类型管理**
   - 可自定义班次类型（如白班、夜班、休息等）
   - 设置班次时间、颜色等属性

2. **排班计划管理**
   - 为不同人员制定排班计划
   - 支持轮班模式设置

3. **请假管理**
   - 记录员工请假信息
   - 支持不同请假类型

4. **节假日管理**
   - 管理法定节假日
   - 支持自定义节假日

5. **多视图日历**
   - 单人日历视图
   - 多人日历视图（重点功能）
   - 当月休假日历

## 高级特性

1. **数据导入导出**
   - 支持JSON格式数据导入
   - Excel报表生成功能

2. **统计分析**
   - 提供统计分析仪表板
   - 支持多种报表生成

3. **自动备份**
   - 自动数据备份功能
   - 支持手动备份和恢复

4. **系统托盘**
   - 支持最小化到系统托盘
   - 右键菜单操作

## 多人日历视图特色

这是程序的重点功能：
- 以网格形式同时显示多人的排班安排
- 每个日期格子可显示当天多个人员的班次
- 支持班次颜色编码和图例显示
- 可按班次类型筛选
- 支持导出当月考勤表到Excel

## 用户界面

- 现代化的UI设计
- 支持亮色/暗色主题切换
- 响应式布局
- 快捷键支持
